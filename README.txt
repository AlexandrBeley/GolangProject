# GolangProject
Можно использовать + - / * без скобок и пробелов. 
Для запуска нужно запустить main01.go, для ввода выражения http://localhost:1229/?nm=1+2  вывод - id запроса
Для получения результата обработки http://localhost:1229/get/?id=0
Для получения всех сведений об обработке http://localhost:1229/data/
Для изменения времени операций http://localhost:1229/times/?timePlus=300ms&timeMinus=100s&timeDivis=15s&timeMult=11s
timePlus - время сложения, timeMinus - время вычитания, timeDivis - время деления, timeMult - время умножения
